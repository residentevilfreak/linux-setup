# linux-setup

my personal scripts and notes for my linux installs.

## usage

### run the script

1.  **clone the repo**

    ```bash
    git clone https://github.com/residentevilfreak/linux-setup ~/Desktop/Setup
    ```

2.  **run the scripts**

    change into the directory

    ```bash
    cd ~/Desktop/Setup
    ```

    then, run the desired script:

    > personal note for me: adjust scrollback in konsole before running cos.sh

    | System | Command |
    | :--- | :--- |
    | **cachyos** | `bash cos.sh` |
